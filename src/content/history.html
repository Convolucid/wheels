The initial version of this page is just a three-section layout with some icons at the bottom. The first section, Home, describes the basic intent of the page. This is the second section, History, which will detail the progress from iteration to iteration. The third section, Process, records the steps I take to create the site from the ground up each time. It's basically a tutorial I made for myself to practice the same coding 'song' over and over. There are currently only four icons for the primary tools I'm using: HTML5, CSS3, Javascript, and Webpack - not counting Git, Vercel, VSCode, and Illustrator, which I'll likely include next. Additional icons for Markdown, Blender, other Adobe Suite programs, Three.js, React, et cetera will be added in future versions of the site.
